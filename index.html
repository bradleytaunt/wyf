Watch Your Figure

An extremely simplistic CSS "plugin" that toggles the figcaption for individual figure elements.

  • Pure CSS - zero JavaScript
  • Only 2.8 kB
  • Fully responsive
  • Only one class name

The icons used for the toggle are base64-based, so it is very easy to swap them out with your own. Feel free to experiment and break this as much as possible!

Why make this?

Boredom, mostly. I'm also obsessed with doing as much "fancy" web things as I can without using any JavaScript. If you're interested, I have a blog where I write about design, CSS and mockup tutorials every once in a while.

The HTML

<figure class="wyf">
    <img src="moose.svg" alt="Moose">
    <input type="checkbox">
    <figcaption>

    <figcaption>
</figure>
